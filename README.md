# OSRSRecipes.com

OSRSRecipes.com is a full-stack web application built for Old School RuneScape players to explore and interact with in-game recipe and item data. The project mirrors real production systems and focuses on scalability, clean architecture, and cloud deployment.

**Live site:** https://osrsrecipes.com

---

## Tech Stack

- Java
- Spring Boot
- React
- AWS Lambda
- AWS API Gateway
- AWS CloudFront
- MySQL (AWS RDS)
- Firebase Authentication

---

## Project Highlights

- Developed a full-stack application using a Spring Boot REST backend and React frontend
- Designed a serverless backend architecture using AWS Lambda and API Gateway
- Deployed a scalable MySQL database on AWS RDS
- Integrated Firebase authentication for secure user login
- Delivered a globally distributed frontend using AWS CloudFront
- Implemented production-style configuration and deployment workflows

---
